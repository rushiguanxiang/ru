# 《如语言：如是架构指令集规范 V3.0》
## （Ru-Shi Architecture V3.0）

**核心理念**  
视觉即物理，观象即执行。  
摒弃一切二进制、十六进制与英文字母的冗余转译，以阴阳二爻直映八位总线电平，以卦象跃迁直控硬件状态。  
**阳爻（⚊）＝高电平/通路，阴爻（⚋）＝低电平/断路。**  
无需编译，无需中间表示，所见即所得。

**关键升级（V3.0）**
- **128条指令**：64本象 + 64易象，全部可执行
- **卦象即指令**：汉字仅为辅助注解，核心是指令符号
- **分层拓扑**：一阳/一阴线循环，二阳/二阴网络，三阳/三阴矩阵
- **本态/易态双模**：本态为预备态，易态为触发态，均可执行不同语义

## 第1章 总线与状态基元

### 1.1 八位总线定义

总线宽8位，从左至右（高位至低位）：

| 线号 | bit7 | bit6 | bit5 | bit4 | bit3 | bit2 | bit1 | bit0 |
|------|------|------|------|------|------|------|------|------|
| 名称 | 恒零 | 上爻 | 五爻 | 四爻 | 三爻 | 二爻 | 初爻 | 枢位 |
| 属性 | 阴(0) | 数据线 | 数据线 | 数据线 | 数据线 | 数据线 | 数据线 | 阴阳枢 |

- **bit7（恒零）**：恒为阴，系统接地或片选使能。
- **bit6～bit1（数据线）**：对应六爻，阳为高电平(1)，阴为低电平(0)。
- **bit0（枢位）**：指令脉冲位。本态为阴(0)，易态为阳(1)。

**机器码生成规则**：
```
本态码 = 0 + 六爻(上→下) + 0
易态码 = 0 + 六爻(上→下) + 1
```
64卦自然生成64个唯一本态码 + 64个唯一易态码 = **128条指令**。

### 1.2 本态与易态（双模执行）

- **本态**：象符无标记，bit0=0，**预备执行态**。  
  例：䷦（蹇，数据准备/地址锁定）
- **易态**：象符右上角标记「一」，bit0=1，**触发执行态**。  
  例：䷦一（蹇，数据写入/指令执行）

**本态与易态均可执行，语义不同**：
- 本态䷦ = 停机/等待（数据未就绪）
- 易态䷦一 = 停机唤醒（数据就绪后继续）
- 本态䷾ = 加法预备（操作数加载）
- 易态䷾一 = 加法执行（结果写入寄存器）

```

         ↕
        ䷢(1,3) ⟷ ䷬(2,3)
         ↕           ↕
        ䷳(1,4) ⟷ ䷦(2,4)
         ↕           ↕    ◄─┐
        ䷃(1,5)      ↕         ↕
         ↕           ↕         ↕
        ䷚(1,6)      ↕         ↕
         ↕           ↕         ↕
        ䷂(2,6) ⟷ ䷜(2,5)    ↕
         ↕           ↕         ↕
        ䷲(3,6) ⟷ ䷧(3,5) ⟷ ䷽(3,4)
         ↕           ↕
        ䷣(4,6) ⟷ ䷭(4,5)

        ䷒(5,6)
```

**拓扑操作定义**：
- **↕ 纵压**：同下阳位，上阳位递减（地址下压）
- **⟷ 横移**：同层相邻节点，移位寄存器操作

## 第3章 编程范式

### 3.1 语法铁律

- **没有变量声明**，象即地址与数据。
- **没有分号**，「一」即状态切换。
- **没有嵌套括号**，拓扑跃迁符 ⟷ ↕ ◄─┐ 即控制流。
- **128条指令全部可执行**，本态预备，易态触发。

### 3.2 代码示例：硬件端口轮询

```plaintext
䷢        // 指针定位到晋态（输入端口）
䷢一 览   // 易态触发，读入数据
䷦        // 指针定位到蹇态（基准值）
䷦一 感   // 易态触发，异或比较
䷽        // 指针定位到小过态（等待）
䷽一 过   // 易态触发，空操作等待
䷜        // 指针定位到坎态（输出端口）
䷜一 明   // 易态触发，写入数据
䷲        // 指针定位到震态（中断线）
䷲一 明   // 易态触发，拉高中断
```

### 3.3 代码示例：拓扑网络寻址

```plaintext
䷓        // 根节点
↓
䷢        // 左支
⟷
䷬        // 右支
↓
䷳        // 下压
⟷
䷦        // 横移
◄─┐
䷂        // 长程闭环，直达远端
```

### 3.4 代码示例：24消息时钟周期

```plaintext
// 从冬至开始，阳气逐级增长
䷗一 → ䷒一 → ䷊一 → ䷡一 → ䷪一 → ䷀一
// 阳极必反，阴气逐级增长
→ ䷫一 → ䷠一 → ䷋一 → ䷓一 → ䷖一 → ䷁一
// 完成一个周期，回到起点
```
