## 编码铁律

```
0x00      = 保留/空/未定义/异常入口
0x01-0x40 = 本象（64卦，序号1-64）
0x41-0x80 = 易象（64卦，序号65-128）
```

## 机器码结构

```
[7]   = 恒零（接地）
[6:1] = 六爻（上爻→五爻→四爻→三爻→二爻→初爻）
[0]   = 乄位（0=本态，1=易态）
```
